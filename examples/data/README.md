
# Example data

- `manufactured_void_image.npy`: deterministic synthetic 3D boolean image (`True` = void)
  used in the PoreSpy extraction notebook.
- `manufactured_porespy_network_*.npz/h5`: generated by notebook `04_mwe_manufactured_porespy_extraction.ipynb`
  when run in the `interop-viz` environment (requires `porespy`).
- `ketton/ketton_segmented.tif`: segmented Ketton carbonate image used in
  `06_mwe_real_porespy_extraction.ipynb`.
- `syn_vugs/synthetic_vugs.tif`: grayscale synthetic vug image used in
  `07_mwe_synthetic_vug_case.ipynb`.
